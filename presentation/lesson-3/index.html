What is it again?

Cascading Style Sheets

- We're going to go through a couple examples, then we are going to recreate a design with HTML and CSS in Codepen

With CSS you can...

  • Add colours to elements and content
  • Add backgrounds colours and images to elements
  • Change the layout and positioning of elements
  • Change the size of elements and content
  • Add animations and transitions
  • Add minor interactivity such as hover styles

The Cascade

  • External Stylesheet
  • Embedded Stylesheet
  • Inline Styles

- CSS holds a lot of power because of something called inheritance

- Inheritance basically means that there is a priority order to where you CSS is, and higher priority will overwrite lower priority

(next slides)

- Three types of stylesheet, in this order of priority

- We're going to look at a diagram, and then go into examples because it sounds more complicated than it is...

- Codepen is effectively an External Stylesheet (demonstrate embedded)

- While we're learning, I don't want us to concentrate too much on all of these three types of styling. We're gunna use Codepen for a bit so it makes it really easy for us.

- In general, in modern web dev, we only ever really use External stylesheets.
